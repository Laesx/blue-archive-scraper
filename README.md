# blue-archive-scraper
Utility to Scrape the JP assets for Blue Archive.

# Setup
## Building UnityPack
```
setup.py build
```
## Installing UnityPack
```
setup.py install
```
or
```
setup.py install --user
```

# Downloading Models
```
py/getModels.py
```
Downloaded Models located at ./downloaded_resource

Assets (Spine and Audio) located at ./assets
